Installation Munee can be installed in 3 easy steps with the use of Composer.

Step 1

Use Composer to install the required libraries

Add "meenie/Munee": "*" to composer.json

If you don't already have a composer.json file, just create a file called composer.json, and insert the following code.

{
    "require": {
        "meenie/Munee": "*"
    }
}

If you don't already have Composer installed, run the following command.

$ curl -s http://getcomposer.org/installer | php

Now to install Munee.

$ php composer.phar install

Step 2

Setup the dispatcher

Create a file in your webroot called munee.php and insert the following code.

<?php
// Include the composer autoload file
require 'vendor/autoload.php';
// Echo out the response
echo \Munee\Dispatcher::run(new \Munee\Request());

Step 3* - Optional

Create a RewriteRule

Open your .htaccess file in webroot and paste the following code.

#### Munee .htaccess Code Start ####
RewriteRule ^(.*\.(?:css|less|scss|js|coffee|jpg|png|gif|jpeg))$ munee.php?files=/$1 [L,QSA]
#### Munee .htaccess Code End ####

If you don't already have a .htaccess file then create one in your webroot and make sure to add RewriteEngine On at the top of the file.

*This step is actually optional and Munee can run without it. Using a RewriteRule makes it easier to use though and is recommended.

Handling Stylesheets Compiling LESS/SCSS, Combining, and Minifying

All LESS & SCSS (SASS is not supported!) files are automatically compiled into CSS and cached, there is nothing extra that you need to do. Any changes you make to your CSS, even LESS/SCSS files you have @imported, will have it's cache automatically recreated, the client side cache invalidated, and the browser will be forced to download the newest version.

One Request For Multiple Stylesheets

If you would like to cut down on requests to the server, then you can combine all of your styles in a comma delimited list

<link rel="stylesheet" href="/css/libs/bootstrap.min.css,/css/site.less">

Minifying CSS

By passing the minify=true option, you can tell Munee to minify all the css by removing all whitespace and putting all the rules on one line.

<link rel="stylesheet" href="/css/libs/bootstrap.min.css,/css/site.scss?minify=true">

Customize Slider

If you want to change the slider's speed transition, open up the file scripts.js and change the following code: 

/* ---------- @ SlidesJS -----------*/
$('#banner').slides({
   preload: true,
   generateNextPrev: true,
   autoHeight: true,
   effect: "slide",
   play: 5000
});
